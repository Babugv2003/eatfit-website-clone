# 🍽️ EatFit Website Clone

A responsive front-end clone of the EatFit website built using HTML and CSS.

---

## 🚀 Project Overview

This project is a static clone of the EatFit website created to improve my front-end development skills.  
The focus was on layout structuring, styling, and responsive design using pure HTML and CSS.

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Positioning
- Responsive Design Techniques

---

## ✨ Features

- Clean and modern UI
- Structured webpage layout
- Responsive design
- Navigation bar
- Styled sections (Hero, Services, Footer, etc.)
- Reusable CSS styling
